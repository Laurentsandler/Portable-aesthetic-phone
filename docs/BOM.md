## Estimated Total Cost

**~$60** (excluding 3D printing materials)

## 3D Printed Parts

The following parts need to be 3D printed (STL files located in the `CAD` folder):

| Part | File | Notes |
|------|------|-------|
| Main Case Body | `Case.stl` | Dark blue PLA recommended |
| Button | `Button.stl` | Same material as case |

### Recommended Print Settings

- **Material:** PLA (dark blue recommended)
- **Supports:** Not required
- **Infill:** 20% or higher

## Where to Buy

- **Screws:** Local hardware store or Amazon
